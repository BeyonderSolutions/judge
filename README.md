# ⚖️ judge
Lazily review Python project inconsistencies and vulnerabilities.

## Installation
```bash
pip install git+https://github.com/BeyonderSolutions/judge.git
```

## Usage

### Running on current working directory

```bash
judge
```

### Running on another directory

```bash
judge /path/to/dir
```
